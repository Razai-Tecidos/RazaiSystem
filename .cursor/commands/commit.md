---
name: commit
description: Gera mensagem de commit a partir das alterações (git diff). Acionado pelo usuário via /commit.
---

Analise o `git diff` (ou as alterações indicadas), resuma o que foi feito e sugira uma mensagem de commit curta e clara, em português quando for o padrão do projeto.

Formato sugerido: tipo(escopo): descrição em imperativo (ex.: feat(api): adiciona endpoint de login).
